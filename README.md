# 🏠 House Price Analysis and Prediction

## 📌 Project Overview
This project analyzes the factors affecting house prices and builds machine learning models to predict house prices using data analytics techniques.

---

## 📊 Dataset
- Dataset: House Prices – Advanced Regression Techniques (Kaggle)
- Contains information about residential houses
- Target Variable: SalePrice

---

## 🧹 Data Preprocessing
- Handled missing values using mean and default values
- Converted categorical variables into numerical format
- Cleaned and prepared the dataset for analysis

---

## 📈 Exploratory Data Analysis (EDA)
- Analyzed distribution of house prices
- Studied relationship between area and price
- Used correlation heatmap to identify important features

---

## 🔍 Key Insights
- Larger houses tend to have higher prices
- Overall quality strongly affects house price
- Garage and number of rooms influence price

---

## 🤖 Models Used
- Linear Regression
- Random Forest Regressor

---

## 📊 Results
- Random Forest performed better than Linear Regression
- Achieved higher R² score
- Provided more accurate predictions with lower error

---

## 🛠️ Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
